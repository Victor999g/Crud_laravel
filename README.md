## Crud de cursos hecho con laravel

El siguiente proyecto cuenta con 3 tablas cursos, estudiantes, 
y lecciones, las cueles tienen diferentes relaciones entre ellas, la base de datos que se utilizo fue MySQL.

2. Clonar el repositorio -git clone https://github.com/Victor999g/Crud_laravel.git
